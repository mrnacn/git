# 2026-06-08

## 批量检查交互升级（L3+）
- 支持格式新增 PDF（accept=".txt,.docx,.html,.pdf"，提示文案同步）
- 点击"开始检测"后，隐藏"继续添加"链接（`batchAddMoreInline`）
- 检测完成后"取消"按钮文本改为"返回"（`cancelBtn.textContent='返回'`）
- 文件清单升级为 table 表头：文件名称 / 文件大小 / 文件格式 / 触发规则 / 风险等级 / 操作
- 触发规则字段在检测完成后填入随机规则（通过='-'，其他=抽样 BATCH_RULES）
- 操作列"查看"按钮调用 `openBatchFileDetail(idx)`，复用意识形态审查详情 Modal
  - 通过文件显示绿色"检测通过"提示卡
  - 风险文件显示高亮图例 + 风险描述（含文件名/大小/格式/触发规则）

## 批量检查交互进一步升级（L3+，第二轮）
- 文件清单表格下方新增独立"继续添加文件"按钮区域（`id=batchAddMoreBlock`）
  - 添加文件后显示，点击"开始检测"后隐藏，点击"返回/取消"后重置
  - 表头内联链接 `batchAddMoreInline` 统一隐藏，改由下方按钮区域承担
- 最近批量检查记录表格新增"操作"列，三条静态示例行均带"查看"链接
  - 新增 `openBatchRecordDetail(batchId, timeStr, total, h, m, l, snapshot)` 函数
  - 新增 `asub-batch-detail` 子页面（含"返回"按钮、批次元信息卡、完整文件清单表格）
  - 新增 `showBatchSubpage(page)` 切换批量检查页与详情页
  - 新增 `batchDetailSnapshots` 对象存储每次检测完成后的文件快照（深拷贝）
  - 动态生成批次记录行时，"查看"按钮调用 `openBatchRecordDetail`（传入快照 batchId）
  - 静态示例批次没有快照时，自动模拟生成文件列表用于展示
  - 批次详情页文件行的"查看"按钮复用 `openBatchFileDetailObj` 弹出意识形态审查 Modal
